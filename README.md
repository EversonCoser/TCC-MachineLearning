# 🏥 APLICAÇÃO TÉCNICA DE APRENDIZADO DE MÁQUINA NA PREDIÇÃO DE MORTALIDADE HOSPITALAR EM PACIENTES ACOMETIDOS POR AVC HEMORRÁGICO - UM ESTUDO DE CASO MIMIC-IV

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) e tem como foco a aplicação técnica de aprendizado de máquina na predição de mortalidade hospitalar em pacientes acometidos por AVC hemorrágico.

A ideia central do trabalho é utilizar dados clínicos para identificar pacientes com maior risco de óbito, permitindo que profissionais da saúde possam tomar decisões mais rápidas e eficazes no tratamento.

A análise foi realizada utilizando o banco de dados MIMIC-IV, um dos maiores datasets públicos de registros clínicos utilizados em pesquisas médicas e de inteligência artificial.

---

## 🎯 Objetivo

O objetivo deste trabalho é desenvolver modelos de aprendizado de máquina capazes de prever a probabilidade de mortalidade de pacientes, com base em dados clínicos e hospitalares coletados nas primeiras 24 horas após a admissão na unidade de saúde.

Essa abordagem pode auxiliar na:

* identificação precoce de pacientes em estado crítico
* priorização de tratamentos e recursos médicos
* apoio à tomada de decisão clínica

---

## 🗂 Dataset

O projeto utiliza o MIMIC-IV (Medical Information Mart for Intensive Care), um banco de dados amplamente utilizado em pesquisas científicas na área da saúde.

Ele contém registros clínicos anonimizados de pacientes atendidos em unidades de terapia intensiva, incluindo informações como:

* sinais vitais
* exames laboratoriais
* histórico clínico
* diagnósticos
* procedimentos médicos

Esses dados permitem o desenvolvimento de modelos preditivos voltados à saúde e medicina baseada em dados.

---

## 🧠 Metodologia

O projeto seguiu as seguintes etapas:

1. Extração de dados utilizando SQL
2. Limpeza e transformação dos dados
3. Análise exploratória dos dados (EDA)
4. Seleção e engenharia de atributos
5. Treinamento de modelos de Machine Learning
6. Avaliação dos modelos
7. Interpretação dos resultados
